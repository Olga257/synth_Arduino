# synth_Arduino

Учебный проект для вуза. Минимальный набор функций синтезатора.
Файл Syn - схема устройства с Tinkercad, файл project - код проекта.
Реализованы:
- извлечение звука при нажатии на кнопки
- смена видов волн (delay, tremolo)
- смена октав вверх и вниз
- запись мелодии в EEPROM
- проигрывание заранее написанной мелодии
- проигрывание записи из EEPROM
